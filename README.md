# SafetyWatch AI — Landing Page

Bu repo, **SafetyWatch AI** bitirme projesinin tanıtım sitesini içerir.

**Canlı site:** [SafetyWatch AI](https://ibrahimbinbuga.github.io/SafetyWatchWebsite/) &nbsp;|&nbsp; **Uygulama:** [work-safety-backend.vercel.app](https://work-safety-backend.vercel.app/)

---

## Proje Hakkında

SafetyWatch AI, inşaat sahalarında ve endüstriyel alanlarda kamera akışlarını gerçek zamanlı analiz ederek iş güvenliği ihlallerini saniyeler içinde tespit eden yapay zeka destekli bir platformdur.

**Tespit edilen ihlaller:**
- Baret eksikliği
- Güvenlik yeleği eksikliği
- Düşme tespiti

**Teknoloji stack'i:**

| Katman | Teknoloji |
|--------|-----------|
| ML Motoru | YOLOv11 (Ultralytics) |
| Backend | FastAPI (Python, async) |
| Web Frontend | React 19 + Vite |
| Mobil | Flutter 3.9 |
| Veritabanı | PostgreSQL + SQLAlchemy |
| Bildirim | Firebase Cloud Messaging |
| Gerçek Zamanlı | WebSocket |
| Dağıtım | Docker + Render.com / Vercel |
